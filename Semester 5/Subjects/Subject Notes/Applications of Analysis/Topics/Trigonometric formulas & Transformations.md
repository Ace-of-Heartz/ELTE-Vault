# Trigonometric formulas
- $\sin(-x) = -\sin(x)$
- $\cos(-x) = \cos(x)$

## $\sin * \cos$
- $\sin(\alpha+ \beta) =\sin \alpha *\cos \beta + \cos \alpha *\sin \beta$ 
- $\sin(\alpha - \beta) = \sin \alpha * \cos \beta - \cos \alpha * \sin \beta$
- $\implies$
- $2*\sin \beta*\cos \alpha =\sin(\alpha + \beta) - \sin(\alpha - \beta)$
